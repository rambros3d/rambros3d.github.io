The Dragon Trucks is an open-source **3D printable model automobile ecosystem** in S-scale (1/64), which makes it compatible with the **Dragon Railway ecosystem** S-scale (1/64). This project is focused on creating a highly **detailed and accurate scale model** of heavy vehicles and automobiles.

## Fully 3D Printed

As we say **"If it can be 3D printed, it will be 3D printed";** From the body and chassis to the wheels and tires, **every major component is 3D printed**. The only additional parts you'll need to purchase are a few screws, bearings, and the necessary electronics.
