## Decide your Scale

Dragon Railway project officially provides models for these scales:

- **S scale** - 1/64
- **G scale** - 1/32

S scale is straightforward as it is the original scale of this project. G scale was later added due to popular request from the community.

Read this guide if you want to do G scale: [G scale Guide](https://docs.google.com/presentation/d/1d4tVA4C70BrevyvksC_FduzbMSn0mg9u2v1Qw9SBJdw/)
