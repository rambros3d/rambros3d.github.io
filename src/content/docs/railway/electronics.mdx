## Official Control Boards

These custom control boards make wiring easy, even for absolute beginners. Minimal soldering required.

The motors, LEDs and batteries have clearly labelled connectors.

:::note
All the required electronic parts are provided with the kit. Only the battery is not included due to shipping restrictions. Please source it from a domestic supplier.
:::

## ESP32 DIY Wiring

:::caution
Only for advanced users.
:::

**Parts required:**

1. ESP32 dev board
2. Motor driver (DRV8833 or L293D)
3. LED lights
4. Resistors (220 ohm)
5. 7805 regulator (AMS1117 - 5V can also be used)
6. N20 Motor (already in the Loco Bogie)
7. Battery

:::warning
**Battery remarks:**

1. Compatible Voltage: 6V to 9V
2. Suitable batteries: 9V rechargeable battery, 2S Li-Fe(6.6V), 2S Li-ion(7.4V) - note that lithium based batteries require BMS/Balance chargers
3. Don't use Lithium batteries without BMS
4. Choose the battery which is comfortable for you. 9V rechargeable battery can be the best option for beginners
:::
